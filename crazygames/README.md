# COIL — CrazyGames launch kit

Everything for publishing COIL on CrazyGames lives here.

| File | What it is |
|---|---|
| `positioning.md` | Competitive read vs **Sticky Orbit** + how COIL differentiates and wins the same audience |
| `store-page.md` | Dashboard fields: title, tags, descriptions, controls, mobile-homepage checklist |
| `asset-prompts.md` | Ready-to-paste image-generation prompts for the **3 thumbnail variants** (+ screenshots) |

**Build status:** launch-ready for Basic Launch (SDK, ads, cross-device save, perf
scaling, reduced-motion, self-hosted fonts all shipped). The remaining work is
external: generate the thumbnail from `asset-prompts.md`, then run an on-platform
QA pass.

**One-line identity (use everywhere):**
> A tiny cosmic creature you fling between glowing gates to climb an endless void —
> one perfect tap at a time. Collect characters, chase the daily route, escape the rising dark.
